Cookie banners and buttons

The GOV.UK programme is working on a new cookie banner design. It currently looks a bit like this.

Tell us whether you accept cookies

We use cookies to collect information about how you use GOV.UK. We use this information to make the website work as well as possible and improve government services.

{{ govukButton({ text: "Accept all cookies", classes: "govuk-!-margin-right-1 govuk-!-margin-bottom-0" }) }} {{ govukButton({ text: "Reject all cookies", classes: " govuk-!-margin-bottom-0" }) }} Set cookie preferences


They would like to differentiate the two buttons more, so that people can more easily distinguish between the positive and negative options.

But, they worry that this treatment is too extreme...

Tell us whether you accept cookies

We use cookies to collect information about how you use GOV.UK. We use this information to make the website work as well as possible and improve government services.

{{ govukButton({ text: "Accept all cookies", classes: "govuk-!-margin-right-1 govuk-!-margin-bottom-0" }) }} {{ govukButton({ text: "Reject all cookies", classes: "govuk-button--white govuk-!-margin-bottom-0" }) }} Set cookie preferences


Instead, they'd like to do something more like this...

Tell us whether you accept cookies

We use cookies to collect information about how you use GOV.UK. We use this information to make the website work as well as possible and improve government services.

{{ govukButton({ text: "Accept all cookies", classes: "govuk-!-margin-right-1 govuk-!-margin-bottom-0" }) }} {{ govukButton({ text: "Reject all cookies", classes: "govuk-button--bordered govuk-!-margin-bottom-0" }) }} Set cookie preferences


These buttons also feature in GOV.UK Publisher.

We considered adding them to the Design System, but there's little evidence that they are needed elsewhere.

It's also not clear how they relate to the grey secondary buttons - are they a replacement or an alternative? If they're an alternative, when should they be used?

The cookie banner will be in the Design System and on every page of GOV.UK, so we don't want to add something without very clear guidance on it's use.
